SapEM Setup Guide

Follow these steps to install and configure your SapEM monitoring system.

1. System Overview

SapEM sensors monitor pipeline pressure continuously and transmit data wirelessly to your system dashboard. Each system typically includes:

  • Pressure sensor nodes
  • Gateway receiver
  • Dashboard software
  • Battery powered sensor modules

2. Installing Sensors

Install sensors at key points within your pipeline system. Recommended locations include:

  • Near compressors
  • At major branch points
  • At pressure regulation stations
  • At pipeline endpoints

3. Powering the Sensors

Insert the battery pack into the sensor module. Once powered, the sensor will begin transmitting data to the system gateway automatically.

4. Connecting the Gateway

Connect the SapEM gateway to your network or monitoring computer. Once connected, the gateway will begin collecting pressure readings from nearby sensors.

5. Viewing the Dashboard

Log into your SapEM dashboard to view:

  • Live system pressure
  • Sensor status
  • Alert notifications
  • System history
