SapEM System Setup Guide

Follow these steps to install and activate your SapEM wireless monitoring system.

1. Install the Sensor Node

Mount the SapEM sensor enclosure near the pressure port and connect the tubing. Ensure the enclosure is properly sealed and protected from moisture.

2. Install the Battery

Insert the lithium battery pack and verify the LED indicator flashes briefly.

3. Power the Hub

Connect the SapEM hub to power and ethernet or WiFi. The hub will automatically scan for nearby sensor nodes.

4. Verify Sensor Registration

Log into the SapEM dashboard and confirm the sensor appears in the device list.
