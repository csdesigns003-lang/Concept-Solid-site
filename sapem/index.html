How SapEM Works

Wireless Sensor Nodes

SapEM nodes collect system data and transmit readings over long-range radio communication.

Gateway Hub

The SapEM hub receives sensor data from nodes and forwards readings to the monitoring dashboard.

User Dashboard

Users view sensor data from the connected web app.
